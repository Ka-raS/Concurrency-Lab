# Concurrency Lab

Basic concurrency concepts: threads, synchronization, semaphores, the Dining Philosophers problem.

## Requirements

- [Python 3](https://www.python.org/downloads/)
- [Jupyter Notebook](https://pypi.org/project/notebook/)

## How To Run

1. Clone Repo
   ```bash
   git clone https://github.com/Ka-raS/Concurrency-Lab.git
   cd Concurrency-Lab
   ```

2. Install Jupyter Notebook:
   ```bash
   python -m venv .venv
   source .venv/bin/activate # .\.venv\Scripts\activate for Windows
   pip install notebook
   ```

3. Open Lab's Jupyter Notebook:
   ```bash
   jupyter notebook lab.ipynb
   ```
